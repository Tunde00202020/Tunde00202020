### 🛸 Alien Invasion — Panda3D
> 3D alien invasion shooter defending a war-torn city — all geometry in pure Python

- Detailed player spaceship with engine trails and cyan glow lighting
- AI wingman that fights beside you and takes damage
- Flying saucer enemies in 4 formation types with dynamic green point lights
- Boss waves every 5 waves with 3 health phases (enrages at 50% and 25% HP)
- Heat-seeking missiles with target lock system (press F)
- 5 power-up types: Rapid Fire, Shield, Screen Bomb, Score Multiplier, Missiles
- Minimap radar showing all enemies in real time
- Real-time kill feed with points earned
- Combo and streak scoring system with S/A/B/C grade on game over
- Storm weather events every 3 waves
- War-torn city environment with burning buildings, smoke, alien tractor beams
- Auto-shader pipeline, atmospheric fog, 4x anti-aliasing
- Zero external assets — every polygon generated in Python code

**Engine:** Panda3D (Python) — covers Panda3D / MonoGame / PyGame bracket

🔗 [View Repository](https://github.com/Tunde00202020/Alien-invasion-panda3d)

### 🏎️ Racing Game — Unity 6
> 3D racing game with real car physics on a professional oval race track

- Real car physics using Unity WheelCollider system
- Professional Cartoon Race Track Oval environment
- Checkered start/finish line
- Minimap showing full track with player position blip
- Speed HUD (km/h), lap counter, race timer, position tracker
- Nitro boost system (Left Shift)
- AI competitor cars with waypoint navigation
- Race Manager tracking race positions (1st, 2nd, 3rd)
- Speed-based camera FOV adjustment for sense of speed
- Kenney car kit models (race car)

**Engine:** Unity 6 LTS (C#) — covers Open 3D Engine (O3DE) / Stride bracket
